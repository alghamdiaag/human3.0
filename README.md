# Human3.0

Human3.0 is an identity-led life transformation system.

It is designed to:
- diagnose how a person’s life is currently operating
- identify the true constraint across four dimensions
- define a clear direction
- build a minimal execution system
- maintain alignment through structured follow-up

---

## Core Concept

The system operates across four dimensions:

- Mind (thinking and perception)
- Body (energy and execution capacity)
- Spirit (meaning and alignment)
- Vocation (real-world output and value)

It identifies:
- patterns
- contradictions
- the primary point of leverage

Then builds:
- a report (10+ pages)
- a 30-day system
- an accountability loop

---

## How It Works

1. Guided interaction (stages)
2. Data extraction (identity, dissonance, dimensions)
3. Synthesis (primary lever selection)
4. Reorientation (direction)
5. Execution design
6. Report generation
7. Accountability system

---

## Structure

The system is modular and agent-based.

Key components:

- docs/ → system logic and definitions
- schemas/ → structured data models
- .claude/agents/ → specialized agents
- .claude/commands/ → execution commands
- prompts/ → system-level prompts

---

## Usage

1. Start a new case using:
   /start

2. Continue using:
   /resume

3. Generate report using:
   /report

4. Weekly check using:
   /review

---

## Important Rules

- The system controls the process
- No stage can be skipped
- Only one primary lever is selected
- Execution must remain minimal
- All outputs must be actionable

---

## Goal

The goal is not insight.

The goal is:
- clarity
- direction
- consistent execution
- measurable change
